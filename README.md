# 📱 Responsive Navbar (Day 3 / 21 Challenge)

A responsive navigation bar that adapts seamlessly across all screen sizes. Built using HTML, CSS, and JavaScript with a mobile-friendly hamburger menu.

---

## 🎯 Goal

To create a navbar that works perfectly on all devices and screen sizes.

---

## ✨ Features

- 📐 Fully responsive design  
- 📱 Mobile-friendly layout  
- 🍔 Hamburger menu for small screens  
- 🖥️ Clean desktop navigation  

---

## 🔄 Project Flow

1. User opens the website  
2. Navbar adjusts based on screen size  
3. On small screens, hamburger menu appears  
4. User clicks menu icon  
5. Navigation links are displayed  

---

## 🧠 Logic Used

- CSS media queries for responsiveness  
- JavaScript toggle functionality  
- DOM manipulation  

---

## 🛠️ Tech Stack

- HTML  
- CSS  
- JavaScript
